# sleep-working-memory-study
Sleep & Academic Performance
Investigating whether sleep duration affects college students' GPA as a proxy for working memory.
Dataset
Student Insomnia and Educational Outcomes — 791 students, 16 variables, cleaned to 750 observations and 9 variables.
Methods
Cleaned and encoded self-reported survey data, then produced four visualizations: scatter plot (sleep vs GPA), histogram (sleep distribution), box plot (GPA by sleep quality), and bar graph (sleep hours for below-average students).
Findings
Weak positive correlation between sleep duration and GPA. Most students slept 7–9 hours. GPA appears influenced by more than sleep alone.
Stack
Python · Pandas · NumPy · Matplotlib
